How To Deploy Laserfiche Apps to SharePoint Sites

  1. Click Here to go to the SharePoint Admin Center or find the same link at learn.microsoft.com. If you can't access this webpage, then you need to ask an Administrator for help.
  2. In the navigation menu on the left, select the "More features" item.
  3. Open "Apps".
  4. Click Upload and select the solution file. If you don't have the solution file, then clone the SharePoint Repo and follow the instructions in the README.md for building the solution.
  5. In your SharePoint Site (Not the Admin Center), navigate to your site’s App catalog by clicking on the "Site Contents" item in the navigation bar on the left side of the page.
  6. Open the "New" Dropdown menu by clicking on the "+" icon.
  7. Select the “App” Menu item.
  8. Add the App named “laserfiche-sharepoint-integration-client-side-solution” if it is present in the list of Apps you can add, or look for it in the SharePoint Store.
  9. Enable the app if you are asked to do so, and disregard any warnings about Microsoft Teams.
  10. Return to your site. Under the “Site Contents” tab, you should now see the SharePoint integration app.

How To Use Laserfiche Apps on SharePoint Pages

  • The Repository Access Web Part:

    1. In your SharePoint Site, select the "Pages" item in the navigation bar on the left side of the page.
    2. Create and open a new site page by clicking the blue "+ New" button and selecting "Site Page" from the dropdown.
    3. Title the page “LaserficheSpApp”.
    4. Move your cursor just below the title area to the white space beneath. This should reveal a hidden "+" button. If you hover over it, it should display the message "Add a new web part in column one”.
    5. Click on that button and Search for “Repository Access”.
    6. Click on the search result with a white L on an orange square. The Repository Access WebPart should now appear on your Page.
    7. Before using the Webpart, make sure to Register Your App in the Laserfiche Developer Console
    8. After you register your App and give it the right redirect URI, you should be able to log in and use the component. For Documentation on how to use the components, reference the User Documentation

  • The Admin Configuration Web Part:

    • Follow the same steps as above, but title the Page “LaserficheSpAdministration”, and add the “Admin Configuration” web part instead of “Repository Access".

  • The Save-To-Laserfiche Web Part:

    • Follow the same steps as above, but title the Page LaserficheSpSignIn, and add the “Save to Laserfiche” web part.

How to Register Your App in the Developer Console

  1. Open the Developer Console
  2. Attempt to Create a New App from Manifest, and copy-paste the manifest provided here.
  3. If the attempt fails because an app with that client ID already exists, find the app with that client id by opening the following url in a new tab: https://app.laserfiche.com/devconsole/apps/{your_client_id_goes_here}/config, where the part in brackets should be replaced by the client_id you copied.
  4. One way or another, an app with that client ID should now exist. Open the app in devconsole and switch from the general tab to the authentication tab.
  5. Add the url of your SharePoint Page with the Laserfiche Web Part as a new redirect URI.
  6. You should now be able to Sign In.
